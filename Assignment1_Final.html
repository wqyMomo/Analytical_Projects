Introduction¶

This report mainly describes the data analysis of the provided general social survey data in New Zealand.The provided dataset covers socio-economic data in New Zealand from the early 1980s. It is an uncleaned, unanalyzed, unpredicted dataset. Therefore, the analyzing processes below include data cleaning, exploratory data analysis, and data presentation.

Goals¶

Looking through all the topics and variables in the dataset, I decided to investigate and discuss the following:

The changes in New Zealand's unemployment rate over the years and some features that it might present.

These goals are basically achieved from the limited time series data.

Exploratory Data Analysis¶

Target variable: unemployment_rate¶

In this part, I will analyse the changes of unemployment rate over the years. Also, I will try to find the relationships between the unemployment_rate and the following variables:

  • 60_64_years_unemp_rate: represents the unemployment rate in 60-64 years old.
  • 65years_and_over_unemp_rate: represents the unemployment rate in 65+ years old.
  • long_term_unemployment_rate: represents the long term unemployment rate.
  • labour_market_insecurity: represents the insecurity of larbour market.
  • underemployment_rate: represents the under-employment rate.

Assumptions:

  • In general, the changes in the unemployment rate in 60+ years old would be similar to the unemployment rate.
  • The changes of the Long-term unemployment rate would be also quite similar to the unemployment rate but it would appear some other features.
  • It might have some relationships between labour market's insecurity and the under-employment rate.

Note: Because the description of each variable is not clear enough, the value of the default unemployment rate is not included the long-term unemployment rate, the unemployment rate between 60-64 years old, and the unemployment rate over 65 by default.

Handle missing value: There are 3 missing values in the unemployment_rate variable, which are in the years 1984, 1985, and 2018. These values do not affect the whole structure of data, so I drop these values directly. The workforce market had not undergone any structural or wind direction changes.

ACF plot and moving average¶

I am interested in the correlation of an unemployment rate at time point t with the unemployment rate at the time before time t. I want to know how long ago the unemployment rate would affect the current unemployment rate.

I used the Auto-correlation Function to find the answer. Autocorrelation measures the relationship between a variable's current value and its past values.
